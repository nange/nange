- [[Rust]]使用`workspace` 模式开发项目，最外层的`Cargo.toml`文件内容需要定义为：
  ```toml
  [workspace.package]
  version = "0.1.0"
  edition = "2021"
  publish = false
  
  [workspace]
  members = [
      "src/some_name",
      ...
  ]
  
  [workspace.dependencies]
  ```
- 都是以`workspace`开头，否则会报错
- 里层的一个`crate`，就正常定义为`[package]`, `[dependencies]`，但是需要包含：`[[bin]]`或者`[lib]`，如：
  ```toml
  [[bin]]
  name = "httpie"
  path = "httpie/main.rs"
  doctest = false
  
  
  ```
